# ftmanlive

Design and implementation of a Django application with User and ActivityPeriod models and to populate the database with some dummy data, and design an API to serve that data in the json format.


## Live View of App:
#### Json Rsponse: https://manftlive.herokuapp.com/
#### API View : https://manftlive.herokuapp.com/status/


## to add dummy Data

> run command:

 ``` python manage.py dummyseed no of user ```
 
``` eg: python manages.py dummyseed 10```
 
 ```it will create 10 user and random activity periods```
